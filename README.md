# ✨ Premium Tic Tac Toe

A sleek, modern, and highly aesthetic Tic Tac Toe game built with vanilla HTML, CSS, and JavaScript. Featuring a stunning glassmorphism design, vibrant gradients, and smooth animations.

![Tic Tac Toe Preview](https://via.placeholder.com/800x450/0b1020/ffffff?text=Premium+Tic+Tac+Toe+Preview)

## 🌟 Features

- **Modern Aesthetics**: Utilizes glassmorphism with backdrop filters and glowing effects.
- **Dynamic Background**: A subtle, animated gradient background that feels premium.
- **Smooth Animations**: Includes pulse effects, hover transitions, and winning animations.
- **Responsive Design**: Works perfectly on both desktop and mobile devices.
- **Interactive UI**: Real-time turn indicator and score tracking.

## 🚀 Getting Started

To run this project locally, follow these simple steps:

1. **Clone the repository**:
   ```bash
   git clone https://github.com/VanshJasuja-1602/TicTacToe.git
   ```
2. **Open the project**:
   - Simply open the `index.html` file in your favorite web browser.
   - OR use an extension like "Live Server" in VS Code for the best experience.

## 🛠️ Built With

- **HTML5**: Semantic structure.
- **CSS3**: Custom properties, Flexbox/Grid, and glassmorphism.
- **JavaScript**: Game logic and state management.
